Is Domain Legitimate

            

            
 This section checks the legitimacy of the domain by analyzing its expiration date.
                The domain is considered legitimate if the time remaining until its expiration is above a certain threshold.
                The current threshold is set to 365 days (1 year). If the domain's expiration date is more than a year in the future,
                it is deemed legitimate. Adjusting the expiration threshold allows for customizable validation criteria.
            

        
            
Is Certificate Valid

            

            
 This section checks the validity of the SSL certificate associated with the analyzed website.
                The SSL certificate is considered valid if the current date falls within the specified validity period,
                which is determined by the 'Valid From' and 'Valid Until' dates obtained from the certificate information.
                If the current date is after the 'Valid From' date and before the 'Valid Until' date, the certificate is deemed valid.
                This ensures that the website's SSL certificate is within its designated active period.
            

        
            
Is HTTPS

            

            
 This section determines whether the analyzed website uses the secure HTTPS (Hypertext Transfer Protocol Secure) or not.
                HTTPS is a secure version of HTTP, providing encrypted communication between the user's browser and the website, enhancing security and protecting sensitive information.
                The result indicates whether the website uses HTTPS, contributing to the overall security of the user's browsing experience.
